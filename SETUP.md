# Инструкция по настройке проекта

## Шаг 1: Установка зависимостей

Выполните команду в корне проекта:

```bash
npm install
```

Это установит все необходимые пакеты:
- `express` - веб-фреймворк
- `pg` - клиент PostgreSQL
- `dotenv` - загрузка переменных окружения
- `bcryptjs` - хеширование паролей
- `jsonwebtoken` - работа с JWT токенами
- `cors` - обработка CORS запросов
- `nodemon` - автоматическая перезагрузка при разработке

## Шаг 2: Настройка переменных окружения

Создайте файл `.env` в корне проекта на основе `.env.example`:

```bash
cp .env.example .env
```

Заполните переменные:
- `DB_HOST`, `DB_PORT`, `DB_NAME`, `DB_USER`, `DB_PASSWORD` - параметры подключения к PostgreSQL
- `JWT_SECRET` - секретный ключ для JWT (обязательно измените в production!)
- `PORT` - порт, на котором будет работать сервер (по умолчанию 3000)

## Шаг 3: Подготовка базы данных

Убедитесь, что PostgreSQL установлен и запущен. Создайте базу данных:

```sql
CREATE DATABASE finance_tracker;
```
