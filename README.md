# Учёт личных финансов

Веб-приложение для учёта и анализа личных финансов пользователей.

## Описание проекта

Система предоставляет пользователям возможность:
- Регистрироваться и проходить аутентификацию
- Создавать финансовые счета
- Учитывать доходы и расходы
- Классифицировать операции по категориям
- Получать агрегированные финансовые данные
- Безопасно хранить информацию в базе данных

## Технологический стек

- **Backend**: Node.js + Express.js
- **База данных**: PostgreSQL
- **Аутентификация**: JWT (JSON Web Token)

## Архитектура

Проект использует модульный монолит с REST API:
- Stateless backend
- Модульная структура (auth, users, accounts, categories, transactions)
- Разделение на слои: routes → controllers → services → repositories

## Установка и запуск

### Требования
- Node.js (версия 14 или выше)
- PostgreSQL (версия 12 или выше)

### Шаги установки

1. Клонировать репозиторий
2. Установить зависимости:
   ```bash
   npm install
   ```
3. Настроить переменные окружения (скопировать `.env.example` в `.env` и заполнить)
4. Запустить миграции базы данных
5. Запустить приложение:
   ```bash
   npm start
   ```
   или в режиме разработки:
   ```bash
   npm run dev
   ```
